## 使用方法  
1.（小白都会的方法）从发布页下载安装包和.NET运行库  
2.小白无脑式安装，安装包纯中文无英文  
3.点击快捷方式即可打开，无需多余步骤！  
4.选择功能大全，开始使用吧~
